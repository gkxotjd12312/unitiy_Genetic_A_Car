# unity_Genetic_A_Car
#인공지능 # 2D유니티 # 유전알고리즘 # 자율주행


## 📜 프로젝트 소개 
AI초입부분을 공부할때 알게된 유전알고리즘으로 가상 공간에서 간단한 자율주행 모델을 만들어봤습니다.
[다른 사람들이 만든거를 보고 저도 저만의 방식으로 하나 만들어 보고싶어서 라이브러리 없이 직접 하나하나 만들어 봤습니다.]

## ⌚ 개발 기간
* 2021.02 ~2021.05

## 👨🏿‍🤝‍👨🏿 제작 맴버
 - 하태성 : 가상 공간 제작, DNN모델 제작, Genetic학습알고리즘 제작

## 🛠 개발 환경
- C#     
- Unity
